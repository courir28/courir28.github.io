Hugo

  1. 安装Git和Go

  2. 安装Hugo

    注：有些主题需要安装extended版本。

    下载解压Hugo，将hugo添加到环境变量中，添加完成后打开Git Bash输入

    hugo version

    检查是否安装成功。

  3. 生成博客文件夹

    打开Git Bash输入

    hugo new site "文件夹名称"

  4. 主题

    Git clone/下载主题文件（官网主题）至themes文件夹中。

  5. 配置

    修改博客文件夹中的config.toml文件或者从themes/主题名/exampleSite文件夹中复制config.toml文件。

  6. 渲染

    打开Git Bash输入

    hugo server

  7. 托管至Github

    建立xxx.github.io仓库，修改博客配置文件，修改baseURL为仓库地址，添加行publishDir= "docs"，打开Git Bash输入hugo，然后：

    cd docs
git init
git remote add origin  https://github.com/xxxxx/xxxx.github.io.git
git add .
git commit -m "first commit"
git pull orgin master
git push -u origin master

    在Github仓库的settings中将Github Pages source改成docs

    参考：

    浅谈我为什么从 HEXO 迁移到 HUGO

    [Hugo+github搭建个人博客（windows10）

Hexo

  1. 创建GitHub Pages仓库

    新建仓库，username.github.io，勾选Initialize this repository with a README

  2. Hexo安装
    按照官方文档

  3. 建站

    克隆博客仓库到本地，cd到仓库文件夹并依次执行以下命令：

    $ hexo init
$ npm install
$ hexo g
$ hexo s

    这样搭建好了本地博客，浏览器输入localhost:4000查看。

  4. 部署博客到GitHub上

    安装hexo-deployer-git

    $ npm  install hexo-deployer-git --save

    修改博客文件夹内的_config.yml文件

    # Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
type: git
repository: git@github.com:username/username.github.io.git
branch: master

    然后

    $ hexo d

  5. 备份博客

    创建新的分支

    $ git  checkout -b hexo(new branch name)

    (注：博客根目录.gitignore文件为不需要备份的文件)

    $ git add .
$ git commit -m "backup"
$ git push origin hexo

    将hexo设置为默认分支

    git 分支相关命令

    a. 查看远程分支

     $  git branch -a

    b. 查看本地分支

    $ git branch

    c. 创建并切换分支

    $ git checkout -b branchname

    d. 删除远程分支

    $ git push origin --delete branchname

    e. 删除本地分支

    $ git branch -d branchname

    f. 同步分支

    $ git fetch

    g. 删除远程分支缓存

    $ git remote show origin
$ git remote prune origin

  6. 恢复博客

    克隆博客（hexo分支）

    克隆下的文件下执行以下命令

    $ npm install hexo-cli
$ npm install
$ npm install hexo-deployer-git

  7. 图片显示问题

    hexo-asset-image插件

    安装方法

    1.0.0版

    npm install hexo-asset-image --save

    0.0.5版

    npm install https://github.com/xcodebuild/hexo-asset-image.git

    两个版本映射关系不同
